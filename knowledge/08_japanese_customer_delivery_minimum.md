# 顧客提供の最小要件（日本語）

このファイルは、テンプレートを顧客へ渡す際の最低条件を日本語で整理したものです。

## 顧客提供の最小要件は4点です

1. ライセンス（`LICENSE`）が同梱されていること。
2. CI（lint + test）が有効で、main ブランチで直近成功していること。
3. Docker 起動手順（`docker compose up --build`）が README に記載されていること。
4. セキュリティ方針（権限モード、sandbox、機密情報の扱い）が文書化されていること。

## 納品前の確認質問

- 「このテンプレートはどの認証モードに対応していますか？」
- 「添付ファイルの 403 はどう切り分けますか？」
- 「ナレッジ検索はどのフォルダを対象にしますか？」

これらに一貫した回答ができれば、説明可能性の観点で合格です。
